# Code of Conduct

We are committed to a welcoming and inclusive community. All participants are expected to:

- Be respectful and considerate
- Assume good intent and engage in constructive feedback
- Avoid harassment, discrimination, or offensive behavior
- Use inclusive language

Unacceptable behavior includes (but is not limited to):
- Harassment, threats, or discrimination
- Gratuitous sexual language or imagery
- Trolling, personal or political attacks
- Publishing othersâ€™ private information without consent

## Scope
This Code applies within all project spaces and when an individual is representing the project in public.

## Enforcement
Report unacceptable behavior via a GitHub issue or by contacting the maintainers. The maintainers may take appropriate action, including warnings, temporary bans, or permanent bans from project spaces.
