# Cloud Cost Handbook

A comprehensive, community-driven guide to understanding and optimizing cloud infrastructure costs.

## Who is this for?

- **DevOps Engineers** managing cloud infrastructure budgets
- **FinOps Practitioners** implementing cost optimization programs
- **Engineering Leaders** trying to understand their cloud spend
- **Platform Teams** building internal cost management tooling

## Topics

### Fundamentals
Learn the basics of cloud billing, cost allocation, and tagging strategies.

### AWS-Specific Guides
Deep dives into optimizing costs for EC2, RDS, S3, Lambda, EKS, and more.

### Strategies
Proven approaches like right-sizing, spot instances, auto-scaling, and multi-account strategies.

### Tools
How to use CloudPulse, AWS Cost Explorer, and CQL (CloudPulse Query Language) for cost analysis.

## Contributing

This handbook is open source. Contributions are welcome via pull requests on [GitHub](https://github.com/cloudpulse/handbook).
